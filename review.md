# 03.:
### Typecasting:
* A way of changing a value from one type to another
* Syntax
  * `(datatype) value;`
* Example:
* 1. `int sum = 3;`
  2. `int count = 2;`
  3. `double average = sum ((double) count);`

# 04.:
### Range Based Loop:
```
for (datatype varname : array) {
    // varname is set to each successive
    // element in the array
}
```
